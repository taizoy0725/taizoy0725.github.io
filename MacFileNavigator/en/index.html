NinjaFileNavigator

A Mac file manager with a familiar Windows Explorer-style workflow

Download on the App Store

A Clearer Way to Manage Files on Your Mac

NinjaFileNavigator brings the sidebar, address bar, tabs, and file list together in one macOS window. Move quickly between frequently used folders and keep files and folders organized with familiar controls.

Requires macOS 14 or later

Key Features

Multiple Tabs

Open folders in separate tabs and switch between them while preserving each tab's location and navigation history.

Quick Access

Use the sidebar to reach your home folder, Desktop, Documents, Downloads, and other frequently used locations.

List and Icon Views

Choose the view that suits your work, adjust icon size, and show or hide hidden files.

File Operations

Create folders, rename, copy, cut, paste, move, compress, and move items to the Trash.

Search and Quick Look

Filter the current folder by name and press Space to preview a file with Quick Look.

iCloud Drive

Browse and manage permitted locations in iCloud Drive, including Desktop and Documents.

File Access

NinjaFileNavigator operates within the Mac App Sandbox. When a protected folder is needed, the app accesses only locations that you explicitly authorize through the macOS folder selection panel.

File listing and operations take place on your Mac. NinjaFileNavigator does not send file contents to its own external server.
